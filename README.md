# EduTrack – Student Performance System

##  Project Overview
EduTrack is a Python-based Academic Performance Management System built using Object-Oriented Programming concepts. It allows users to manage, analyze, and rank student records efficiently.

##  Features
- Add / Update / Remove Students
- Grade Calculation
- Class Topper Identification
- Student Ranking System
- Grade Distribution
- Subject-wise Analysis
- Class Statistics
- Input Validation (Error Handling)

## Technologies Used
- Python
- OOP (Object-Oriented Programming)
- Lists & Dictionaries
- Lambda Functions
- Sorting & Searching
